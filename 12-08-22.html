1. Difference between flex row and column

Ans:

  • Flex row will arrange flex items in a row(horizontally) which mains horizontal axis as main axis and verical axis as cross-axis.
  • Flex row will arrange flex items in a columns(vertically) which mains horizontal axis as cross axis and verical axis as main-axis.

2. Explain inline , internal and external stylesheet

Ans:

Inline:
If we want to give styling to a particular tag element inside it's opening tag, it is known as Inline styling
Internal:
If we want to give styling to a particular tag element inside the html file in head section under style tag, it is known as Internal styling
External:
If we want to give styling to a particular tag element outside the html file and link that styling sheet to html using link tag in head section is called External Styling

3. Justify-content allows you to do what?

Ans:
Justify-content allows to arrange the flex-items in required manner on main axis. Values which will be taking by Justify-content are:

  • flex-start
  • flex-end
  • center
  • space-evenly
  • space-around
  • space-between

4. Difference between absolute and relative positioning

Ans:

Absolute:
"Position: absolute" is used to give the offsets from the first non-static ancestor and element's space will be occupied by another element.
Relative:
"Position: relative" is used to give the offsets from the original position of the element and element's space will be reserved so that no other element will take it.

6. What is the z-index in CSS?

Ans:
Z-index simply can be defined as visibility index. By default every element has z-index value as 0. We can give both positive and negative values for z-index depending on our requiement.

7. What is the difference between padding and margin?

Ans:

Padding:
Padding is used to create a space around content-area. Color of padding will always be the background color of the content-area. We cannot change the color of padding
Margin:
Margin is used to create a space to our targetted element from other elements.

8. What is box sizing?

Ans:
Box-sizing is attribute which is used to define the box sizes. It usually take 2 values namely:

  • Content-box
  • Border-box
Content-box is the default value of box-sizing. It this the padding,border and margin will be added to the given width and height, whereas in border-box they are included in the given width and height.

12. Explain CSS box model

Ans:

  • CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties.
  • By inspecting the element we will be able to distinguish content-area, padding, border and margin of an element.

13. what is the difference between display none and visibility hidden?

Ans:

  • Both of them are used to hide the content in the browser.
  • The major difference is that "display:none" means the space provided for that element on the browser will be occupied by the next element
  • And "visibility:hidden" means the space provided by the browser for that element remains reserved for that element only.
