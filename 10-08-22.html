1. Difference between div and span

Ans:
Div: It is a block-level element,which takes whole width of the browser

Example: Div element
Span: It is an inline element,which only takes the width upto its content area. Example: span element

2. Difference between class and IDs

Ans:
Class: Class is an attribute which is used to select a specified tag. A tag can have multiple class values(Class names) and a single class name(Class value) can also be given to multiple tags. For targetting the selected tag we will use ".classvalue" which can also be said as ".classname" 

            Example:
           1. < i class="first second third fourth"> content < /i>
           2. < p class="test"> content < /p>
              < div class="test"> content < /div>
              < span class="test"> content < /span>
ID ID is an attribute which is used to select a specified tag. ID cannot take multiple values and every ID must be unqiue, a same ID value(ID name) cannot be assigned to a different tag. For targetting the selected tag we will use "#IDvalue" which can also be said as "#IDname" 
            Example:
            1. < i id="test1"> content < /i>
               < div id="test2"> content < /div>

5. Difference between block and inline

Ans:

  1. Block elements are the elements who take the entire browser width; where as inline elements only take the width they required for their text content.
  2. Another major difference in them is block level elements always starts in a new line, where as inline elements stays in the same line( if it not a added to block element).
  3. Block level elements dont allow others to occupy their free space(width) as it completely assigned for them, where as inline elements allows other elements.
            Example for block level element:
            
1. < p>< /p>
 
2. < div>< /div>

            Example for inline elements:
            1. < span>< /span> 2. < i>< /i> 3. < u>< /u>

6. Difference between em and strong

Ans:
Em and strong both are used to emphasize the text which is written inside them. Along with emphasizes "em" gives italic style to the font whereas "strong" gives bold style to the font. 

            Example for "em":
            < em> Emphasized Italic Text< /em>
            Example for "strong":
            < strong> Emphasized Strong Text< /strong>

7. What are attributes?

Attributes are properties of a tag. Every attribute is unique. Some attributes take values,which some dont take any values. Attributes which dont take any values are known as boolean attributes. 

            Examples of attributes:
            1. < span width="" style=""> Content area < /span>
              In this width and style are attributes
            2. < video controls autoplay> < /video>
              In this controls and autoplay are boolean attributes

8. Difference between HTML tags and elements

Ans:
HTML tags means only the opening tag and closing tag along with its attributes and their values. 

For Example: < div class="test" style="color:blue; background-color: red;">< /div>
Whereas HTML element means HTML tag along with the content between the tags. 
For Example: < div class="test" style="color:blue; background-color: red;">This is content< /div>

10. What are semantic tags?

Ans:

  • Semantic tags are block-level elements which are used to differentiate the area of display for which we are creating design.
  • Eventhough we change the tag name, it wont show any difference because they all act in a same way. But for the better understanding of the coders(who reads the code or who wants to make further changes) we use semantic tags
            Some of the semantic tags are:
                1. < nav> Content < /nav>
                2. < header> Content < /header>
                3. < section> Content < /section>
                4. < footer> Content < /footer>
